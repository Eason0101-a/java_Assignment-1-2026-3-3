## 數學模型
本程式使用 Euler 方法進行兩種常微分方程模型的數值迭代：

1. 線性模型
$$dx/dt = -k \cdot x$$

2. 非線性 Logistic 模型
$$dx/dt = r \cdot x \cdot (1 - x/K)$$
